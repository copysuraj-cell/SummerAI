# SummerAI

A collection of AI and web projects built over the summer. Each folder is its own self-contained project with its own README and setup instructions.

---

## Projects

### `rag/` — RAG Comparison: FAISS vs Cognee
Streamlit app that runs the same question through two different RAG pipelines side by side — FAISS (vector similarity search) and Cognee (knowledge graph). Shows how retrieval method affects answer quality. Uses Grok via xAI for graph building and Ollama locally for generation.

### `observability/` — Observability Comparison: Logfire vs Arize
Streamlit app that runs the same FAISS RAG pipeline through two different observability backends — Logfire and Arize — so you can compare how each platform surfaces traces, spans, inputs, and outputs from an AI system.
